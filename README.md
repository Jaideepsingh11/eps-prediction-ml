\# EPS Prediction using Machine Learning



This project predicts \*\*Earnings Per Share (EPS)\*\* using financial ratios and machine learning techniques.  

The model is trained using historical financial data and deployed using a \*\*Flask web application\*\*.

\## Features Used

\- ROCE (%)

\- CASA (%)

\- Return on Equity / Networth (%)

\- Non-Interest Income / Total Assets (%)

\- Operating Profit / Total Assets (%)

\- Operating Expenses / Total Assets (%)

\- Interest Expenses / Total Assets (%)



---



\##  Machine Learning Approach

\- Feature selection using \*\*Variance Inflation Factor (VIF)\*\*

\- Feature scaling using \*\*StandardScaler\*\*

\- Model trained using \*\*XGBoost Regressor\*\*

\- Trained model and scaler saved using `pickle`



---



\##  Web Deployment

\- Backend: \*\*Flask\*\*

\- Frontend: \*\*HTML\*\*

\- Real-time EPS prediction through web form

\- Same preprocessing used during training and inference
